Link nokidhungry.org/rebuild

I was tasked to create a micro-site for No Kid Hungry's Fall Campaign. The Fall Campaign is an initiative by No Kid Hungry that used murals to inspire others to learn more about the No Kid Hungry mission. The clients wanted a site that will display all murals on one page. Since I didn't want the murals to be lost within the brand, I kept the design simple and easy to understand. The five murals are displayed in the header for the user to easily jump from one mural to the other.
